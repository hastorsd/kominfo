## Cara Menjalankan Proyek

1. Jalankan git clone + url_repo_ini

2. Lakukan composer install dan npm install

3. Pastikan terkoneksi ke database, dengan cara jalankan php artisan db.

4. Run command: php artisan serve --host=0.0.0.0 dan npm run dev

5. Akses /sales untuk melihat tampilan jawaban soal no 1

6. Akses /api/products dan /api/products/{id} untuk melihat tampilan jawaban soal no 2

7. Akses /products untuk melihat halaman list produk, dan /get-products untuk mengambil data API. untuk jawaban soal no 3

8. Akses /api/categories dan /api/suppliers untuk melihat data api endpoint tambahan
